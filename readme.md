# Avenir Booking API

Backend API sederhana untuk simulasi sistem booking tour menggunakan FastAPI dan PostgreSQL.

## Features

* GET Booking
* POST Booking
* PUT Booking
* DELETE Booking
* PostgreSQL Integration
* Swagger Documentation

## Tech Stack

* Python
* FastAPI
* PostgreSQL
* Psycopg2

## Run Local

```bash
uvicorn main:app --reload
```
